# Street Art Orlando
Explore a gallery of local artwork through the Street Art Orlando app.

## Building
Use Android Studio or the gradle wrapper script.

*Linux*  
`./gradlew assembleDebug`

*Windows*  
`gradlew.bat assembleDebug`

## Requirements
- Android Studio 2.0+
- Google Maps API Key
